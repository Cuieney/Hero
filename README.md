# Super Stitch

#### 视频编辑工具
一款视频超级拼接软件，只要你敢拼我就敢接，打造一款自由拼接视频画面的App 工具，项目目前功能还并未完全完善，需要大家的一致意见来进行完善，所以呢项目博主还在自己努力开发阶段，等项目完善了一定给大家带来源码。大家可以根据下面的二维码 或者链接进行下载App，体验目前版本的功能。功能大部分意见可以用了。


[apk下载地址](https://fir.im/5ary?release_id=5c8f422f548b7a4fcce8199d)
扫二维码也是可以下载的

![downloadapk.png](https://upload-images.jianshu.io/upload_images/3415839-7151f96459c8ad39.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)



### feature
* 视频画面拼接（主要的）
* 视频宽高裁剪
* 视频时长裁剪
* 添加视频边框及修改边框颜色
* 添加视频封面
* 添加视频背景图片
* 添加视频水印
* 添加背景音乐
